# GoFood - Full Stack Food Delivery Application

## Overview
GoFood is a complete MERN stack web application that allows users to browse food items, add them to a cart, and securely place orders. It features a fully functional frontend built with React and a robust backend API powered by Node.js, Express, and MongoDB.

## Tech Stack
* **Frontend:** React.js, Bootstrap
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
* **Security:** JWT (JSON Web Tokens), Bcrypt (Password Hashing)
